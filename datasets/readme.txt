This folder contains the 12 synthetic datasets used in subsection 4.3 of the paper

K.A. Benatti, J.V. Pamplona, L.G. Pedroso and A.A. Ribeiro, An approach to the clustering problem with capacity
constraints.

Usage:
- each file corresponds to a dataset
- the first 500 lines are the dataset itself
- the next 500 entries are the weights of the points
- the last 5 lines represent the capacity of each cluster
